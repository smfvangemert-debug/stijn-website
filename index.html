From Idea to Website

Deze website is ontstaan vanuit mijn interesse om te leren hoe websites worden gebouwd. In plaats van alleen theorie te volgen, wilde ik een project maken waarin ik zelf een website vanaf de basis opbouw.

Het idee achter dit project

Om het leerproces concreet te maken, heb ik besloten deze website te gebruiken als persoonlijke oefening. Hier experimenteer ik met HTML, CSS en later ook JavaScript om stap voor stap te leren hoe een website technisch en inhoudelijk wordt opgebouwd.

Eerdere ervaringen

In het verleden heb ik al ervaring opgedaan met programmeren en technische projecten. Deze website is een manier om mijn kennis verder uit te breiden richting webontwikkeling en om beter te begrijpen hoe moderne websites werken.
