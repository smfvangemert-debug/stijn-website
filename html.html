Wat is HTML?

HTML staat voor HyperText Markup Language en vormt de basis van vrijwel elke website. Het wordt gebruikt om de structuur van een webpagina te beschrijven, zoals koppen, tekst, links, afbeeldingen en formulieren.

HTML is geen programmeertaal maar een markup language. Dat betekent dat het beschrijft hoe informatie op een pagina wordt gestructureerd.

HTML Tags

HTML werkt met zogenaamde tags. Tags staan tussen < > en vertellen de browser wat voor soort inhoud er op de pagina staat.

De meeste HTML-elementen hebben een begin-tag en een eind-tag. 

<p>Dit is een paragraaf</p>

In dit voorbeeld:

  • <p> = opening tag
  • Dit is een paragraaf = inhoud
  • </p> = closing tag

Basisstructuur van een HTML pagina

Elke HTML-pagina heeft een vaste basisstructuur zodat browsers weten hoe ze de pagina moeten lezen. 

<!DOCTYPE html>
<html>
<head>
<title>Mijn pagina</title>
</head>

<body>
<h1>Welkom</h1>
<p>Dit is mijn eerste website.</p>
</body>

</html>

De belangrijkste onderdelen zijn:

  • <html> – bevat de hele webpagina
  • <head> – informatie over de pagina (titel, meta data)
  • <body> – de zichtbare inhoud van de website

Veelgebruikte HTML elementen

  • <h1> – <h6> → koppen
  • <p> → paragrafen
  • <a> → links
  • <img> → afbeeldingen
  • <ul> en <li> → lijsten

Voorbeeld van een link:

<a href="https://example.com">Ga naar website</a>

Waarom HTML belangrijk is

HTML vormt de structuur van een website. Zonder HTML kan een browser niet weten waar tekst, afbeeldingen of knoppen moeten staan. CSS zorgt vervolgens voor de opmaak en JavaScript voor interactie.
