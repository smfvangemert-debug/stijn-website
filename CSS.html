Wat is CSS?

CSS staat voor Cascading Style Sheets. Het is een stylesheet-taal die wordt gebruikt om het uiterlijk en de presentatie van webpagina’s te bepalen. Met CSS kun je bijvoorbeeld kleuren, lettertypes, spacing, positionering en layout aanpassen. HTML bepaalt de structuur van een webpagina, terwijl CSS bepaalt hoe deze structuur er visueel uitziet. [1]

Door CSS in een apart bestand te plaatsen kunnen meerdere webpagina’s dezelfde styling delen. Hierdoor wordt de code overzichtelijker en eenvoudiger te onderhouden. Ook kunnen browsers het CSS-bestand opslaan (cachen), waardoor websites sneller laden. [1]

Hoe CSS werkt

CSS werkt met regels die bestaan uit een selector en een declaration block. De selector bepaalt op welk HTML-element de stijl wordt toegepast. De eigenschappen in het declaration block bepalen hoe het element eruitziet. [2] 

h1 {
  color: blue;
  font-size: 40px;
}

In dit voorbeeld:

  • h1 → selector (selecteert alle h1-koppen)
  • color → property (eigenschap)
  • blue → value (waarde)

CSS Selectors

Selectors bepalen op welke HTML-elementen een CSS-regel wordt toegepast. Er bestaan verschillende soorten selectors, zoals elementselectors, class-selectors en id-selectors. Hiermee kun je heel precies bepalen welke onderdelen van een pagina gestyled moeten worden. [2]

Element selector

p {
  color: red;
}

Alle paragrafen worden rood.

Class selector

HTML: 
<p class="highlight">Belangrijke tekst</p>
CSS: 
.highlight {
  color: orange;
}

Alle elementen met de class highlight krijgen deze stijl.

Het CSS Box Model

Een van de belangrijkste concepten in CSS is het box model. Elke HTML-element wordt door de browser weergegeven als een rechthoekige box. Deze box bestaat uit vier onderdelen: content, padding, border en margin. Deze onderdelen bepalen samen de grootte en positie van een element op een webpagina. [3]

  • Content – de inhoud van het element (tekst of afbeelding)
  • Padding – ruimte tussen content en border
  • Border – de rand rondom het element
  • Margin – ruimte tussen elementen
div {
  margin: 20px;
  padding: 15px;
  border: 2px solid black;
}

Door het box model te begrijpen kun je bepalen hoe elementen op een pagina worden uitgelijnd en hoeveel ruimte ze innemen.

Wat kun je met CSS doen?

Met CSS kun je vrijwel het volledige uiterlijk van een website bepalen. Bijvoorbeeld:

  • Kleuren en achtergronden aanpassen
  • Lettertypes en typografie instellen
  • Layouts bouwen met Flexbox of CSS Grid
  • Responsive websites maken voor verschillende schermgroottes
  • Animaties en hover-effecten toevoegen

Hierdoor vormt CSS samen met HTML en JavaScript een van de belangrijkste technologieën van het moderne web. [1]
