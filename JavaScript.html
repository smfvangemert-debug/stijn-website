Wat is JavaScript?

JavaScript is een programmeertaal die wordt gebruikt om webpagina’s interactief en dynamisch te maken. Samen met HTML en CSS vormt JavaScript de basis van moderne webontwikkeling. HTML beschrijft de structuur van een webpagina, CSS bepaalt de opmaak, en JavaScript zorgt voor gedrag en interactie. [1]

Met JavaScript kunnen ontwikkelaars functionaliteiten toevoegen zoals knoppen die reageren op een klik, interactieve formulieren, animaties, real-time updates en dynamische content die verandert zonder dat de pagina opnieuw hoeft te laden. [2]

Hoe JavaScript werkt

JavaScript wordt uitgevoerd door de browser. Wanneer een webpagina wordt geladen, leest de browser de HTML en CSS om de pagina weer te geven. Vervolgens kan JavaScript de pagina aanpassen door elementen te veranderen, nieuwe elementen toe te voegen of acties uit te voeren wanneer een gebruiker interactie heeft met de website. [1]

JavaScript is een zogenaamde event-driven taal. Dit betekent dat code wordt uitgevoerd wanneer bepaalde gebeurtenissen plaatsvinden, bijvoorbeeld wanneer een gebruiker op een knop klikt of een formulier invult. [3]

Variabelen

Variabelen worden gebruikt om gegevens op te slaan. In JavaScript kunnen variabelen worden gedeclareerd met let, const of var. Moderne JavaScript-code gebruikt meestal let en const. [4] 

let naam = "Stijn";
const leeftijd = 25;

console.log(naam);

Variabelen kunnen verschillende soorten data bevatten, zoals tekst, getallen, arrays en objecten.

Functies

Een functie is een blok code dat een bepaalde taak uitvoert. Functies maken code overzichtelijker en herbruikbaar. [5] 

function begroet(naam) {
  console.log("Hallo " + naam);
}

begroet("Stijn");

Hier wordt een functie gemaakt die een naam als input krijgt en een begroeting op het scherm toont.

De DOM (Document Object Model)

De DOM is een representatie van de HTML-structuur van een webpagina. JavaScript kan via de DOM elementen selecteren en aanpassen. Hierdoor kan de inhoud van een pagina dynamisch veranderen. [6] 

document.getElementById("titel").textContent = "Nieuwe titel";

In dit voorbeeld zoekt JavaScript een HTML-element met het id titel en verandert de tekst.

Events

Events zijn gebeurtenissen die plaatsvinden op een webpagina. JavaScript kan luisteren naar deze gebeurtenissen en daarop reageren. Voorbeelden van events zijn klikken, toetsen indrukken of het laden van een pagina. [3] 

document.querySelector("button").addEventListener("click", function() {
  alert("Je hebt op de knop geklikt!");
});

Wanneer de gebruiker op de knop klikt, wordt een melding weergegeven.

Wat kun je met JavaScript doen?

JavaScript wordt voor veel verschillende toepassingen gebruikt, zoals:

  • Interactie toevoegen aan webpagina’s
  • Formulieren controleren (validatie)
  • Dynamische inhoud laden zonder de pagina te verversen
  • Animaties en interactieve elementen maken
  • Webapplicaties bouwen
  • Browsergames ontwikkelen

Naast gebruik in browsers kan JavaScript ook op servers worden uitgevoerd, bijvoorbeeld met Node.js, waardoor het een veelzijdige programmeertaal is geworden. [2]
